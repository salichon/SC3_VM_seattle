text¶

Text output plugin for scm.

Description¶

The text plugin writes for each client a status text file to a configurable directory. Each text file is named after the client with the extension ".txt".

Plugin¶

The text plugin is installed under share/plugins/monitor/mtextplugin.so.

To add the plugin to scm, add it to the plugin list:

plugins = ${plugins}, mtextplugin

Configuration¶

mtextplugin.outputDir¶

Type: string

Output directory where [client].txt is written to. Additionally an file description.txt will be created to show the order of tags used in the client status file. Default is @LOGDIR@/scm/.
